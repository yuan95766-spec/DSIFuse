# DSIFuse

Official PyTorch implementation of **"Semantic-guided multi-level collaboration fusion network for visible and infrared image fusion"**.

---

## Overview

**DSIFuse** is a deep learning framework for visible and infrared image fusion.  
This repository provides the official PyTorch training code.

---

## Recommended Environment

Please set up the environment with the following recommended package versions:

- `torch==1.13.0`
- `torchvision==0.14.0`
- `kornia==0.7.0`
- `pillow==9.4.0`
- `numpy==1.21.2`

### Install (example)

```bash
pip install torch==1.13.0 torchvision==0.14.0
pip install kornia==0.7.0 pillow==9.4.0 numpy==1.21.2
